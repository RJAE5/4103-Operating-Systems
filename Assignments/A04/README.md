## A04 - c/C++ File Commands
### Rykir Evans
### Description:

This program may prompt the user for a file name and a starting byte to inspect and retrieves 3 consecutive 1-byte characters and outputs them to the console, signifying the initials of a "criminal" being investigated. By default, it will use this file as per assignment instructions.

### Usage
For WSL, the following instructions may be used to run the program:
1. Navigate to the directory of the program
2. Use the g++ compiler or any standard C++ compiler `g++ main.cpp -o main`
3. Run the program using `./main`
4. Follow prompts as given
